# Nexus Forum

**Modular Community Platform for Roblox**

---

## English (Brief)

Nexus Forum is a modular community platform for Roblox.  
It uses a zero-trust architecture with profile-based DataStore sync. No RemoteEvent spam. Clean UI. Discord-ready.

**What’s inside:**
- Loader, Main, Client, Server, Config
- DataServiceTyped for real-time data observation
- Discord Webhook logging and bot integration
- Clean, responsive design system

**Discord:** https://discord.gg/VqmKMbwGnR

**Status:** Active development. Core is stable.

---

## Русский (Кратко)

Nexus Forum — модульная платформа для сообществ в Roblox.  
Построена на архитектуре с нулевым доверием и синхронизацией через профили. Без RemoteEvent-спама. Чистый UI. Готова к интеграции с Discord.

**Внутри:**
- Loader, Main, Client, Server, Config
- DataServiceTyped для наблюдения за данными в реальном времени
- Логирование и интеграция через Discord Webhook
- Единая, адаптивная дизайн-система

**Discord:** https://discord.gg/VqmKMbwGnR

**Статус:** Активная разработка. Ядро стабильно.
